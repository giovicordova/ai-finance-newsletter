# Routines

Prompts that drive the scheduled Claude tasks for this project. Source of truth for what each scheduled trigger should run.

| File | When | What it does |
|---|---|---|
| `news.md` | Mon-Fri 06:00 Europe/Rome | Generates the day's edition via the `ai-finance-briefing` skill, reviews, commits, and sends the news post to Telegram. |
| `take.md` | Friday 07:00 Europe/Rome | Reads today's edition (already generated by `news.md`) and sends only the `Claude's weekly take:` post, one hour after the news. |

If a scheduler prompt drifts from these files, fix the scheduler, not the file. See `docs/SCHEDULING.md` for setup details.
